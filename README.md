# Tetris

[Tetris](https://oshukezu.github.io/Tetris/)
1. Controls use the keyboard arrow keys (up, down, left, right).
2. The A key is used to rotate blocks, and the B key swaps with the next block.
3. The difference between the mobile version and the desktop version is that the mobile version uses virtual on-screen buttons, while the desktop version is controlled via the keyboard.
4. The game speed will gradually increase as your score goes up.

[俄羅斯方塊](https://oshukezu.github.io/Tetris/)
1. 操作為鍵盤的上下左右
2. 轉動是Ａ鍵，與Ｂ鍵與下一個互換
3. 手機版與電腦版的差異是，手機版有虛擬按鍵，電腦版控制請透過鍵盤
4. 速度會隨著分數逐漸上升
